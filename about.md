A visual coaching tool for building and presenting animated soccer plays. Record custom plays by dragging players and the ball across a proportionally accurate pitch, capture keyframes, and play them back as smooth animations. Organize plays by age group (U6 through Adult), add descriptions and coach notes for teaching points, and step through each phase at adjustable speeds. Everything saves locally in the browser. Built as a single standalone HTML file — just drop it on any web host or open it directly.
